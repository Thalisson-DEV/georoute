Central de Ajuda

Tire suas dúvidas sobre como utilizar o Sipel Logistics Helper.

Como Buscar Clientes?

A barra de busca na tela inicial é inteligente e detecta automaticamente o tipo de dado inserido.

  • ✓ Instalação: Digite o número completo.
  • ✓ Conta Contrato: Digite o número da conta.
  • ✓ Medidor/Série: Digite o número de série do equipamento.
  • ✓ Poste: Digite o código do poste (ex: P00123).

Planejador de Rotas

Crie rotas otimizadas para suas equipes economizarem tempo e combustível.

  • 1. Adicione clientes à lista de roteirização clicando no botão "Adicionar à Rota" nos resultados de busca.
  • 2. Acesse a aba "Roteirização" no menu.
  • 3. Selecione a Equipe responsável.
  • 4. Escolha o ponto de partida (Base ou Localização Atual) e clique em "Gerar Rota".

Histórico de Rotas

Consulte rotas realizadas anteriormente para auditoria ou reutilização.

  • ➜ Na tela de Roteirização, clique na aba "Histórico de Rotas".
  • ➜ Selecione a equipe desejada para ver a lista de rotas salvas.
  • ➜ Clique em uma rota para visualizar o trajeto no mapa e os clientes atendidos.

Cadastro e Gestão

Mantenha a base de dados atualizada através do menu "Cadastro".

  • ★ Clientes: Cadastre novos clientes individualmente preenchendo o formulário completo (Instalação, Conta, Coordenadas).
  • ★ Equipes: Gerencie as equipes de campo, definindo setor de atuação e base operacional.
  • ★ Importação: Utilize a opção de "Importar Clientes" para carregar arquivos CSV em massa.
